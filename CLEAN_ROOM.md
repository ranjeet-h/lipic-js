Clean-Room Implementation Disclaimer

This repository is intended to be developed under a clean-room process.

No proprietary source code, confidential materials, or non-public documentation from any third party may be copied into or referenced while implementing this project.

Any implementation work must be based only on:
- Publicly available specifications and documentation
- Original independent design and engineering
- Newly created tests and examples written for this repository

If any contributor has had access to non-public materials related to similar systems, they must not use that knowledge to reproduce protected implementation details in this repository.
